# Lilypad

An open-source prompt engineering framework.

Automatic versioning and tr

## Getting Started

### Tab Completion

`lilypad --install-completion`

### Start Server

`lilypad start`

### Create stub

`lilypad create {FN_NAME}`

### Run llm function

`lilypad run {FN_NAME}`

### Edit function params

`lilypad run --edit {FN_NAME}`
